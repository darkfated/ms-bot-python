# ms-bot-python

Многофункциональный бот для Discord

## Команды

Команды (текстовые и slash):

- /играть или !играть <ссылка/запрос>
- /стоп или !стоп
- /пауза или !пауза
- /продолжить или !продолжить
- /пропустить или !пропустить
- /повтор или !повтор [вкл/выкл]
- /очередь или !очередь
- /сейчас или !сейчас
- /помощь или !помощь
- /пинг или !пинг

## Запуск проекта

Поставьте окружение и перейдите в него

```bash
python -m venv venv
venv\Scripts\activate # Windows
source venv/bin/activate # Linux/macOS
```

Установите зависимости

```bash
pip install -r requirements.txt
```

Настройке конфиг, созданный командой

```bash
cp .env.example .env
```

Запустите исполняемый файл

```bash
python bot.py
```

## Требования

Для работы бота требуется установленный `FFmpeg`
